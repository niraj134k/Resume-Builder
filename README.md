# AI Resume Builder (MERN Stack)

An AI-powered full-stack web application to generate professional resumes effortlessly using smart content suggestions and dynamic templates.

---

## Features

- Generate resume content automatically with AI suggestions (skills, summary, experience).  
- Fully editable and customizable sections.  
- Multiple modern templates for different resume styles.  
- Download resumes as PDF.  
- User authentication and data storage for saving resumes.  
- Responsive design for desktop and mobile.

---

## 🛠️ Tech Stack

* **Frontend:** React.js – Dynamic UI & live preview  
* **Backend:** Node.js + Express.js – API endpoints & AI integration  
* **Database:** MongoDB – Store user data and resumes  
* **AI Integration:** OpenAI / GPT API – Suggest skills, experience, and summaries  
* **Styling:** CSS / TailwindCSS – Modern responsive design

## Clone the repository
git clone https://github.com/your-username/ai-resume-builder.git

## Navigate to the project folder
cd ai-resume-builder

## Install backend dependencies
cd server
npm install

## Install frontend dependencies
cd ../client
npm install

## Set up environment variables
## (replace with your actual credentials)
MONGO_URI=<your-mongodb-uri>
OPENAI_API_KEY=<your-openai-api-key>

## Start the backend server
cd ../server
npm start

## Start the React frontend
cd ../client
npm start

## Open the app in your browser
## http://localhost:3000
